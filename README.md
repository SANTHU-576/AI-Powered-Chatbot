# 🤖 AI-Powered Chatbot

An AI-powered chatbot built using **Python, Flask, NLTK, Scikit-learn, HTML, CSS, JavaScript, and SQLite**. The chatbot understands user queries, provides intelligent responses, and stores conversation history in a database.

---

## 🚀 Features

- 💬 AI-powered chatbot interface
- 🧠 Natural Language Processing (NLTK)
- 🔍 TF-IDF based response matching
- 🗄️ SQLite database for chat history
- 📅 Live Date response
- 🕒 Live Time response
- 🎨 Modern and responsive UI
- ⌨️ Press Enter to send messages
- 🗑️ Clear chat option
- 🤖 Typing animation
- 📚 Multiple predefined intents

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/SANTHU-576/AI-Powered-Chatbot.git
```

Move into the project folder:

```bash
cd AI-Powered-Chatbot
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
